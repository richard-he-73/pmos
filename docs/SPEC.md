# 技术栈约束

## 前端

- **框架**: React/Vue 3
- **UI 组件库**: Ant Design / Element Plus (可根据设计系统定制)
- **图表**: ECharts / Chart.js
- **状态管理**: Redux / Pinia
- **路由**: React Router / Vue Router
- **样式**: CSS Variables + Tailwind CSS

## 后端

- **运行时**: Python（版本3.14，使用uv管理项目环境）
- **框架**: FastAPI
- **数据库**: MongoDB
- **缓存**: Redis
- **消息队列**: RabbitMQ / Kafka

## 部署

- **容器**: Docker + Kubernetes
- **CI/CD**: GitHub Actions / Jenkins
- **监控**: Prometheus + Grafana
